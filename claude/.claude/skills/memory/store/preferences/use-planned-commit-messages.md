# Use Planned Commit Messages Verbatim

When a commit message has been explicitly planned (in a planning section, AGENTS.md, conversation, or otherwise), use that exact message when committing. Do not review past git log messages and substitute a paraphrase or shortened version. The planned message takes precedence over style-matching.
